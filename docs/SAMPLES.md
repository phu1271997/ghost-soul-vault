# Sample Families

Ghost ships with two ready-made evaluation stories for demos and reviewer walkthroughs.

## Family A

Traditional father, three heirs, and highly visible variance in work ethic and social proof.

Use case:
Show how the same vault can approve different heirs differently based on the same persona.

## Family B

Integrity-focused founder family with one philanthropic heir and one scandal-shadowed heir.

Use case:
Show how open-web evidence can lower trust even when an heir still belongs to the family.

## Seeding

Use:

```bash
python3 scripts/seed_demo_family.py <contract-address> --sample family_scenario_1.json
python3 scripts/seed_demo_family.py <contract-address> --sample family_scenario_2.json
```
